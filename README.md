# 🎆 Cyber Fireworks (사이버 불꽃놀이)

방구석에서 즐기는 여름밤 불꽃놀이. 실사 밤바다 배경 위로 픽셀 감성의 불꽃을 터뜨리며 힐링하는 아담한 게임입니다.

Made with **Unity 6 (URP 2D)**.

## ✨ 기능
- 🎆 **불꽃 발사** — 밤하늘을 클릭해 형형색색 불꽃을 터뜨리기
- ✍️ **글자 불꽃** — 원하는 글자(한글/영문)를 불꽃으로 하늘에 쓰기
- 🏖️ **해변 꾸미기** — 모닥불·파라솔·돗자리·별 머리띠 등 픽셀 아이템으로 아늑하게 꾸미기
- 🌙 **NPC 자동 불꽃쇼** — 해변의 아이를 클릭하면 자동으로 불꽃놀이 감상
- ☀️ **낮 모드 이스터에그** — 달을 여러 번 클릭하면 튜브 탄 아이로 즐기는 횡스크롤 점프 미니게임
- 🎵 파도 소리 · 폭죽 효과음 · BGM

## 🎮 조작
- **마우스 클릭** — 불꽃 발사
- **ESC** — 일시정지 / 메뉴
- **낮 모드**: SPACE/클릭 점프, R 다시하기, ESC 나가기

## 🛠️ 개발
